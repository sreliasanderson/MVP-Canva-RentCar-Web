# MVP-Canva-RentCar-Web
##Descrição

RentCar Web é uma plataforma web para locação de veículos que permite o gerenciamento de usuários, veículos e reservas através de uma interface simples e intuitiva.

O projeto foi concebido utilizando práticas de Lean Inception e Scrum para definição do MVP, backlog e planejamento das entregas.

##Objetivo

Disponibilizar uma solução digital para que clientes possam consultar veículos disponíveis e realizar reservas online, enquanto administradores gerenciam a frota e as locações.

##Backlog do MVP
Gestão de Usuários
Catálogo de Veículos
Gestão de Reservas
Gestão da Frota
Plataforma Técnica

##Metodologia
Lean Inception
Scrum
Desenvolvimento Incremental
MVP (Minimum Viable Product)

##Status
Projeto em fase de definição e prototipação do MVP.
